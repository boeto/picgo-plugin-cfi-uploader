### Steps to get apiToken

- [Create Token](https://dash.cloudflare.com/profile/api-tokens) --->
- Get started --->
- Enter your token name --> Permissions -> Select 'Cloudflare Images' -> Select 'Edit' --->
- Create Token --->
- Copy & Save your token
